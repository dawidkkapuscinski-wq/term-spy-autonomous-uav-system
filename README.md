# TERM-SPY

Autonomous UAV/UGV system integrating ROS 2, AI-based object detection, SLAM, LTE telemetry and MAVLink communication.

## Project overview

TERM-SPY was designed as a modular autonomous system for UAV and UGV platforms.  
The project integrates:

- ROS 2
- RTAB-Map SLAM
- YOLO AI object detection
- MAVROS / MAVLink
- LTE communication
- ZeroTier overlay VPN
- Real-time video transmission

## Main technologies

- Ubuntu Linux
- ROS 2 Humble
- NVIDIA Jetson AGX Orin
- ZED 2i stereo camera
- OpenCV
- CUDA
- ffmpeg
- MAVROS
- RTAB-Map

## Features

- AI object detection
- 3D mapping and SLAM
- LTE telemetry
- Autonomous back-to-home system
- Real-time video streaming
- GPS-loss recovery logic
- Overlay VPN communication
